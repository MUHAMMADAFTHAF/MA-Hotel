# Hotel Booking App (Static)

A fully responsive hotel booking website built using **HTML** and **CSS only**.  
This project includes a **Home Page**, **About Page**, and **Contact Page**, designed to showcase a hotel’s services and provide essential information.

---

## 🌟 Features
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile.
- **Home Page**: Highlights hotel offerings and provides navigation.
- **About Page**: Shares details about the hotel and its mission.
- **Contact Page**: Displays contact information and a simple form layout (non-functional, static).
- **No JavaScript / No Backend**: Purely static design for simplicity and easy deployment.

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** for styling and responsiveness
